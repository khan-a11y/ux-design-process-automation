01

한 장 결론

TL;DR

똑같은 경쟁자는 없음. 하지만 단계별로 쪼개면 우리보다 깊은 전문도구가 카테고리마다 존재함. 우리의 자리는 그 사이 빈 칸 — 전 과정 통합 + 완전 무료 + 로컬/프라이버시 + 초급자 교육형. 결론: 우리를 “올인원 대체품”으로 밀면 품질에서 지고, “학습·프라이버시·초안 허브 + 각 단계는 무료 전문도구로 내보내기”로 포지션을 잡으면 경쟁자가 거의 없음.

02

시장 지형 — 단계별로 누가 있나

UX 도구 시장은 “올인원 하나”가 아니라 단계 특화 도구들의 생태계임. 우리 7스튜디오가 닿는 6개 카테고리별 대표 주자와 무료 여부를 정리함.

P1·P2 · 리서치 / 합성

리서치 리포지토리

  • 전사·자동 태깅·테마 추출·인사이트 저장. 이 카테고리가 가장 성숙함.
  • 무료·오픈소스가 가장 약한 영역 → 우리 강점이 살아있는 곳.
Dovetail $29~ Notably $25~ Looppanel $395 Marvin free~ Condens €15~
P3·P5 · 와이어 / UI 생성

AI 와이어프레임·디자인

  • 텍스트→화면 생성, Figma 연동, 코드 export. 성장 가장 빠름.
  • UX Pilot이 우리와 가장 넓게 겹치는 단일 경쟁자.
UX Pilot $0~$31 Uizard free~ Visily free~ Galileo/Stitch
P5 · 핸드오프 / 토큰

디자인 토큰·개발 전달

  • 토큰을 코드(CSS/Tailwind/Swift)로, 스펙 자동 생성.
  • Penpot·Style Dictionary가 완전 무료·오픈소스 → 연계 최적.
Penpot OSS Style Dictionary Tokens Studio free Zeplin
P6 · 검증

사용성·휴리스틱·QA

  • 프로토타입 테스트·접근성 감사. 자동화 툴은 WCAG의 30~40%만 잡음(나머지 수동).
  • 무료 티어·오픈소스가 풍부함.
Maze free-forever Lyssna free axe / WAVE Lighthouse
P7 · 운영

제품 분석·A/B·이상탐지

  • 실트래픽 기반 히트맵·세션리플레이·실험. 우리 브라우저 통계로는 초안까지만.
  • PostHog·Matomo·Clarity가 강력한 무료/오픈 → 실서비스는 이쪽.
MS Clarity 무료 PostHog MIT GrowthBook OSS Matomo GPL
P1 · 전사(로컬)

음성→텍스트 (프라이버시)

  • 업로드 없이 로컬 처리. 우리와 철학이 같은 유일한 진영.
  • 한국어는 SenseVoice·Clova Note가 Whisper보다 유리(띄어쓰기).
Whisper 로컬 SenseVoice Clova Note free Buzz

※ 캡처는 각 서비스 공개 홈페이지를 헤드리스 브라우저로 직접 촬영(2026-07-01). 마케팅 페이지라 제품 UI 전체는 아님 — 성격·포지션 참고용.

03

포지셔닝 맵

두 장으로 판단함. ①우리가 서 있는 빈 칸(범위×개방성), ②전문도구와 왜 싸우지 말고 병행해야 하는지(학습성×실무 깊이).

맵 A · 커버 범위 × 개방성(무료·로컬)

오른쪽 위로 갈수록 “전 과정을 무료·로컬로”. 그 코너는 사실상 비어 있음.

Dovetail·Notably는 좁고 유료(좌하), Penpot·PostHog는 무료·오픈이나 단계 특화(좌상). 우리만 우상단.

맵 B · 학습·자기설명성 × 실무 깊이

우리(좌상)와 전문도구群(우하)은 대각선으로 상보적. 겹치지 않으니 병행이 정답.

우리는 “가르치며 초안까지”, 전문도구는 “깊이·품질”. 사용자를 우리에서 시작시켜 필요한 단계만 전문도구로 넘기면 학습·품질을 둘 다 얻음.

05

우리의 강점과 구조적 약점

확증만 하지 않고 냉정하게 봄. 강점을 키우고 약점은 전문도구로 메우는 게 1번 업그레이드의 논리.

지키는 강점 (경쟁자가 못 채우는 빈 칸)

  • 전 과정 통합 — P1~P7 한 도구. 경쟁사는 전부 단계 특화.
  • 완전 무료·무빌드 — 계정·구독 없이 HTML 더블클릭.
  • 로컬·프라이버시 — 업로드 없이 브라우저 안 처리. 민감 데이터 가능.
  • 교육형·자기설명 — 초급자를 가르치며 진행. 시장은 이걸 “코스”로만 팜(도구 아님).
  • dossier 연계 — 단계 산출물이 한 권으로 이어짐(export 파편화 없음).
  • 리서치 합성 — 무료·오픈이 가장 약한 영역이라 상대적 우위.

구조적 약점 (전문도구로 메울 곳)

  • 정밀 한국어 전사 — 브라우저 무료 모델의 구조적 한계 → 로컬 SenseVoice/Clova.
  • 고fidelity UI·클라이언트 납품물 → Penpot·Figma·UX Pilot.
  • 실참여자 사용성 테스트·패널 → Maze·Lyssna.
  • 실트래픽 A/B·분석·운영 → PostHog·Clarity·Matomo.
  • 팀 협업·공유 리포지토리 없음(localStorage 1인) → Dovetail류·Notion.
  • 7파일 컴포넌트 복제 drift(내부 유지보수 부채).

UX 프로세스 학습·온보딩(초급자 스캐폴딩) · 예산 0 / 개인·소규모 프로젝트 · 업로드 불가한 민감 데이터(프라이버시) · 1차 합성·초안(인터뷰 가이드·어피니티·HMW·와이어 초안) · 단계 빠뜨리지 않는 체크리스트형 워크플로우.

~ 병행 권장

정밀 한국어 전사(→SenseVoice/Clova) · 고fidelity UI·납품물(→Penpot/Figma/UX Pilot) · 팀 리포지토리(→Dovetail/Notion) · 실사용자 테스트(→Maze/Lyssna) · 실서비스 분석(→PostHog/Clarity). 1번 업그레이드가 바로 이 “내보내기 다리”를 놓는 작업.

✕ 단독 부적합

팀 협업이 핵심인 조직의 메인 허브 · 엔터프라이즈 보안·감사 요건 · 대규모 참여자 리크루팅. 이 경우 우리는 “보조·학습·초안” 역할로만.

권고 포지션 — “Dovetail/UX Pilot을 대체하는 올인원”이 아니라 “UX 프로세스를 가르치며 초안까지 만들고, 각 단계는 무료 전문도구로 내보내는 학습·프라이버시 허브”. 이 자리는 시중에 경쟁자가 거의 없음.
07

1번 · 무료 전문도구 연계 업그레이드 플랜

핵심 원리 = “내보내기 다리(export bridge)”. 우리는 초안·구조·학습을 담당하고, 표준 포맷으로 내보내 무료 전문도구가 이어받게 함. 전부 무료·무빌드·file:// 유지. 아래는 임팩트×난이도 순 3파도.
