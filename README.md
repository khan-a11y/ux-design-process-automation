# UX×AX 자동화 도구 모음

> UX 표준 6단계(P1→P6)의 AI 자동화 과제 **29개**를, 흩어진 프롬프트 카드가 아니라
> **연계성으로 묶은 7개 스튜디오 앱**으로 실제 자동화했습니다. **유료 API 없이 무료로 동작합니다.**

- 시작: **`index.html`** 더블클릭 (허브) → 스튜디오 선택
- 작성: DIR4 AI Lab · 문서 ID `UXAX-STUDIO-2026`

---

## 무엇이 다른가 — "프롬프트 칸"이 아니라 "실제 자동화"

각 도구는 LLM 호출 주변의 실제 작업을 **브라우저 안에서 코드로** 처리합니다(전부 무료):

- 🎙️ **오디오 → 자동 전사**(브라우저 Whisper) · 🧮 **임베딩 + k-means 군집**(VOC·어피니티)
- 📄 **PDF 텍스트 추출 + RAG 검색**(데스크 리서치) · 📊 **A/B 통계 실계산**(z검정·p값·신뢰구간)
- 🎨 **WCAG 색대비 자동 계산** · 🖼 **이미지 픽셀 diff**(canvas) · 📈 **z-score 이상탐지**
- 🔀 **Mermaid 플로우 자동 렌더** · CSV/PDF 파싱 · 배치 처리 · 차트 · MD/CSV/JSON 내보내기

LLM은 이 파이프라인의 한 단계(요약·분류·라벨·해석)만 담당합니다.

## 그룹핑 원칙 — 과부하 없이, 연계성으로

29개를 한 앱에 다 넣지 않고, **함께 쓰는 일끼리 앱당 3~5개**로 묶었습니다.
단계(P)를 가로질러 자연스럽게 이어지는 것은 같은 앱으로(예: 디자인 토큰 → 코드 스펙).

| 스튜디오 | 도구 | 묶은 이유 |
|---|---|---|
| 🔍 **리서치** (`P1_리서치스튜디오.html`) | 전사·요약 · 설문코딩 · VOC · 데스크리서치 · 인터뷰가이드 | 데이터 수집·분석 |
| 🧩 **정의** (`P2_정의스튜디오.html`) | 어피니티 · 여정맵 · HMW · PRD | 합성 → 정의 |
| 💡 **아이디에이션** (`P3_아이디에이션스튜디오.html`) | 발산 · IA · 플로우 · 우선순위 | 발상 → 구조화 |
| ✎ **콘텐츠·카피** (`P4_콘텐츠스튜디오.html`) | UX 라이팅 · 다국어 현지화 · 더미 콘텐츠 | 글·콘텐츠(라이팅→현지화) |
| ⌘ **디자인·핸드오프** (`P5_디자인핸드오프스튜디오.html`) | 와이어 · 토큰 · 코드스펙 · 체인지로그 | 디자인 → 개발 전달 |
| ✓ **품질 검증** (`P6_품질검증스튜디오.html`) | 사용성 · 휴리스틱/접근성 · 디자인 QA | 화면 품질 평가 |
| 📊 **피드백·운영 분석** (`P7_피드백운영스튜디오.html`) | 피드백분류 · A/B · 이상탐지 · 운영피드백 | 출시 후 측정·운영 |
| 🎭 **퍼소나** (별도 전용 앱) | 페르소나 · 인터뷰 가능한 AI 사용자 · 합성 | `D:\AI_Project\260618_퍼소나_스튜디오\v0.2` |

각 앱은 상단 **탭**으로 도구를 전환하고, 각 탭 맨 위 **📖 사용법** 가이드(준비물 → 단계 → 알아둘 점)를 제공합니다.

## 💸 무료가 기본 — AI 엔진 4가지 (모든 앱 공유)

설정(⚙️)에서 한 번 고르면 7개 앱이 같은 설정을 씁니다. 앱을 처음 열면 로컬 Ollama/브라우저 AI를 자동 감지합니다.

| 엔진 | 비용 | 설치 | 비전(이미지) |
|------|------|------|------|
| 🌐 브라우저 내장 AI (WebGPU) | 무료 | 없음 | ✕ |
| 🖥️ 로컬 Ollama | 무료 | 1회 | ✕ |
| ✍️ 수동 / 챗GPT(복붙) | 무료 | 없음 | ○(직접 업로드) |
| ☁️ Anthropic API | 유료 | 없음 | ○ |

> 전사·임베딩·통계·픽셀 diff·색대비는 **엔진과 무관하게** 브라우저에서 자동 계산됩니다.
> LLM 단계(요약·분류 등)만 위 엔진을 사용합니다.

## 시작하기

1. **`index.html` 더블클릭** → 스튜디오 선택.
2. (권장) **로컬 Ollama**: `start-ollama.bat` 더블클릭 → 자동 감지. 한국어 모델: `ollama pull exaone3.5:7.8b`
3. 또는 **수동/챗GPT**: 설정에서 선택 → 프롬프트 창을 챗봇에 복붙.
4. (선택) **API**: `config.example.js` → `config.js`로 복사 후 키 입력.
5. `file://`에서 막히면 `serve.bat`(로컬 서버)로 여세요.

## 📋 산출물 연결 — 보관함(Workspace)

7개 앱은 우하단 **📋 보관함** 버튼으로 **하나의 공유 보관함**을 씁니다(앱 간 공유).

- 결과 카드의 **📤 보관함**으로 산출물을 저장 → 다른 도구/앱의 입력칸 위 **📥 가져오기**로 불러와 다음 단계 입력에 합칩니다. (예: P1 인터뷰 요약 → P2 HMW, 아이디어 발산 → P3 우선순위, P1 VOC → P7 피드백)
- 보관함에서 산출물을 **편집·확정**한 뒤 넘길 수 있습니다(AI 제안을 사람이 다듬기). 전체 **JSON 백업**도 지원.

## ✅ 검증(스모크 테스트)

수정 후 빈 화면/오류를 자동 확인합니다.
