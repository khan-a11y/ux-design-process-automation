UX×AX 자동화 교보재

AI로 UX 디자인 프로세스를 어디까지 자동화할 수 있나 — 6단계 29개 과제를 실제로 만들어 본 데모·학습자료. 전부 무료(브라우저·로컬·복붙).

📚 교보재 이건 "AI로 UX 자동화가 어디까지 되나"를 직접 만들어 본 학습자료입니다

UX 표준 6단계의 자동화 과제 29개를 전부 무료로 구현해, AI가 진짜 해내는 부분과 아직 사람 검수가 필요한 부분의 경계를 눈으로 보여줍니다. 완성된 단일 제품이라기보다, 각 단계에서 자동화가 어디까지 닿는지 확인하는 데모 모음입니다. 실무에 바로 쓰는 "흐름 도구"는 별도 프로젝트(260617_UX디자인프로세스_도구)에서 P1~P2에 집중해 따로 만듭니다.

🔢 코드 실집계
숫자·계산을 브라우저 코드가 직접 함(전사·통계·색대비·픽셀 diff·군집·차트). AI 안 거침 → 신뢰 높음.
⚗️ 혼합
코드가 계산하고 AI가 해석·라벨링. 숫자는 믿고, 문장은 검수.
✨ AI 생성
LLM이 글·분류·아이디어를 생성·추측. 틀릴 수 있어 반드시 검수 필요.

스튜디오 (앱) — 연계성으로 묶음

29개 자동화를 모두 한 앱에 넣지 않고, 함께 쓰는 일끼리 앱당 3~5개로 묶었습니다. 엔진 설정은 모든 앱이 공유합니다.

단계별 29개 도구 전체 (P1→P6)

UX 표준 프로세스 순서로 본 전체 목록. 각 카드 아래 칩(🔢/⚗️/✨)은 그 도구가 AI에 얼마나 의존하는지를 보여줍니다 — 교보재의 핵심 관전 포인트. 각 카드는 해당 도구가 들어 있는 스튜디오의 탭으로 바로 엽니다.

🎭 퍼소나 스튜디오 (별도 전용 앱)

페르소나 생성 + 인터뷰 가능한 AI 사용자 + 자동 인터뷰 + 인사이트 합성. P2-A2 / P5-A5는 이 앱이 전담합니다.
